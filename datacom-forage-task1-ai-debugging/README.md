🚀 Project Overview
This repository contains the technical deliverables for the Datacom Job Simulation, focusing on modernizing legacy Python systems using AI-assisted workflows. The primary objective was to transform a buggy, inefficient data processing script into a production-ready, tested, and optimized module.

🛠️ Key Deliverables
Refactored Core (/src): A high-performance Python script (process_data_final.py) optimized for clarity and data processing efficiency.

Test Suite (/tests): Comprehensive unit tests (TEST_CASES.py) designed to identify edge cases and verify the integrity of the final fix.

Engineering Log (/docs): A detailed technical breakdown (DEBUG_LOG.md) documenting the iterative interaction with AI co-pilots and the underlying architectural decisions.

🧠 Engineering Mindset & Methodology
Instead of manual debugging, I utilized an AI-first approach to handle technical debt:

Context-Rich Prompting: Engineered specific prompts to help the AI understand the structure of unfamiliar legacy code.

Iterative Hypothesizing: Treated AI suggestions as hypotheses, rigorously testing and refining them through local unit tests rather than blind acceptance.

Human-in-the-Loop Oversight: Applied critical thinking to verify AI-generated logic, ensuring the final output meets enterprise-grade standards for security and performance.

💻 Tech Stack
Language: Python 3.x

Tooling: AI-Powered IDE (Cursor/GitHub Copilot/VS Code)

Testing: Unittest / Pytest
